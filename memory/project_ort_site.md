Built a full single-page marketing site for Ort Strategy Tech Services.

**Why:** User wanted a beautiful, animated landing page to showcase services and capture leads.

**Stack:** Vite + React, Framer Motion, @emailjs/browser, react-icons, CSS Modules.

**Color palette:** Deep navy (#020818), electric blue (#1d6bf3), cyan (#00c8ff), white.

**Sections:** Navbar (sticky/blur), Hero (particle canvas + orbs), About, Services (4 cards), Stats (animated counters), Projects Carousel (mock browser screens), Testimonials Carousel (auto-rotate), Contact Form (EmailJS), Footer.

**Contact form:** Uses EmailJS. Credentials must be set in `src/components/Contact.jsx` — `EMAILJS_SERVICE_ID`, `EMAILJS_TEMPLATE_ID`, `EMAILJS_PUBLIC_KEY`. Instructions in `EMAILJS_SETUP.md`.

**How to apply:** When resuming work on this project, check these files first for context.
